Question 3

Media queries in CSS allow styles to be applied conditionally based on device characteristics like screen size, resolution, or orientation, enabling responsive web design. They ensure websites adapt seamlessly to different devices, improving user experience, performance, and accessibility. Using the @media rule, developers can define specific styles for various conditions, such as screen widths (max-width: 768px), orientations (orientation: portrait), or resolutions (min-resolution: 2dppx). Media queries are crucial for creating flexible designs, supporting a mobile-first approach, and delivering optimized layouts across desktops, tablets, and smartphones.
